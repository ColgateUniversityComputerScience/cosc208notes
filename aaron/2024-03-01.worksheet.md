## Warm-up

Q1: _Consider the following function that creates a tic tac toe board._


```c
#include <stdlib.h>
#define BOARD_SIZE 3
char **create_board() {
    char **board = malloc(sizeof(char *) * BOARD_SIZE);
    for (int r = 0; r < BOARD_SIZE; r++) {
        board[r] = malloc(sizeof(char) * BOARD_SIZE);
        for (int c = 0; c < BOARD_SIZE; c++) {
            board[r][c] = ' ';
        }
    }
    return board;
}
```
