* Q1: _What is the output of this program?_


```c
#include <stdio.h>
int main() {
    char first[] = "Colgate";
    char second[10] = "Univ";
    printf("%lu\n", strlen(first));
    printf("%lu\n", strlen(second));
    first[strlen(first)] = '-';
    second[strlen(second)-1] = '.';
    printf("%s%s\n", first, second);
    first[3] = '.';
    first[4] = '\0';
    printf("%s %s\n", first, second);
}
```
