## System calls (syscalls)

* Example program


```c
#include <stdio.h>
#include <unistd.h>
int name() {
    int pid = getpid();
    return pid;
}
int main() {
    int p = name();
    printf("I am a process. My name is %d.\n", p);
}
```

    I am a process. My name is 1276.
