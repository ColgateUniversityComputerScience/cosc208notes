## Pointers

* Q8: _What does the following program output?_


```c
#include <stdio.h>
int main() {
    int a = 1;
    int b = 2;
    int *x = &a;
    int *y = &b;
    printf("%d %d %d %d\n", a, b, *x, *y);
    a = 5;
    *y = 7;
    printf("%d %d %d %d\n", a, b, *x, *y);
```

## Pointers as parameters

* Example


```c
#include <stdio.h>
void value(int a) {
    a = 2;
}
void pointer(int *b) {
    *b = 3;
}
int main() {
    int v = 1;
    int *p = &v;
    value(v);
    printf("%d\n", v);
    pointer(p);
    printf("%d\n", v);
}
```
