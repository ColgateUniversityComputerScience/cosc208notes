## Type conversion


```c
#include <stdio.h>
int main() {
    int x = -123456;
    printf("%d %u\n", x, x);
}
```


```c
#include <stdio.h>
int main() {
    unsigned char c = 300;
    printf("%u\n", c);
}
```


```c
#include <stdio.h>
int main() {
    unsigned int i = 300;
    unsigned char c = i;
    printf("%u\n", c);
}
```
