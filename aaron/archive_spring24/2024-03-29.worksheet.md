## Mapping C loops to assembly

* Example mapping while loops to assembly


```c
/* 1*/   #include <stdio.h>
/* 2*/   int pow2(int n) {
/* 3*/       int result = 1;
/* 4*/       while (n > 0) {
/* 5*/           result = result * 2;
/* 6*/           n = n - 1;
/* 7*/       }
/* 8*/       return result;
/* 9*/   }
/*10*/   int main() {
/*11*/    printf("2^10 = %d\n", pow2(10));

```
