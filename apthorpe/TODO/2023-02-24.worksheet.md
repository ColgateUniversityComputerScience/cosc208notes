## Warm-up
* Q1: _What does the following program output?_


```c
#include <stdio.h>
int main() {
    int numsA[] = {1, 2, 4, 8, 16, 32, 64};
    int *numsB = numsA + 3;
    printf("%d %d\n", *numsA, *numsB);
    numsB += 1;
    *numsB += 1;
    printf("%d %d %d %d\n", numsA[2], numsA[3], numsA[4], numsA[5]);
    *(numsB - 1) = *(numsA + 1);
    numsB[1] += 1;
    printf("%d %d %d %d\n", numsA[2], numsA[3], numsA[4], numsA[5]);
}
```
