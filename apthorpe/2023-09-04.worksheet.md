## Warm-up: practice with strings and input

* Q1: _What is the output of this program?_


```c
#include <stdio.h>
#include <string.h>

int main() {
    char first[] = "Colgate";
    char second[10] = "Univ";
    
    printf("%lu\n", strlen(first));
    printf("%lu\n", strlen(second));
    printf("%s %s\n", first, second);
    
    first[2] = '.';
    first[3] = '\0';
    printf("%s %s\n", first, second);
  
    printf("%lu\n", strlen(first));
    first[strlen(first)] = '-';
    printf("%s%s\n", first, second);
}
```
