## Type conversion


```c
// Example 1
#include <stdio.h>
int main() {
    int x = -123456;
    printf("%d %u\n", x, x);
}
```


```c
// Example 2
#include <stdio.h>
int main() {
    unsigned char c = 300;
    printf("%u\n", c);
}
```


